# Work Area

This folder is for local reverse-engineering work.

- `apks/`: downloaded Android APKs and related binaries (ignored by git)
- `captures/`: packet captures and logs (ignored by git)
- `decompiled/`: generated output from `jadx` or `apktool` (ignored by git)
- `notes/`: tracked research notes and protocol findings
- `samples/`: scratch payload samples or exports that should stay local unless curated
